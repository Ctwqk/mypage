# mypage

Personal project site that links demos and publications, plus a small Flask + SQLite chatbot backend.

## Highlights

- Static project-gallery pages for demos and publications
- Small Flask backend for persistent chat sessions and image attachments
- Useful as a lightweight portfolio host rather than a product-scale application

## Tech Stack

- HTML, CSS, JavaScript, Flask, SQLite

## Repository Layout

- `frontpart/projects/index.html`
- `frontpart/publications/index.html`
- `backpart/DeepSeek/server.py`

## Getting Started

- Open the static pages under `frontpart/` directly in a browser or serve them with any static-file server.
- Run `python server.py` from `backpart/DeepSeek/` if you want the chatbot backend.
- Treat the repository as a personal showcase workspace rather than a unified production app.
